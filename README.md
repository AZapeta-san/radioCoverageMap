# radioCoverageMap

An interactive web application designed to visualize radio network and station coverage throughout the United States. The platform provides a dynamic map interface that enables users to explore coverage areas, search for radio networks, and analyze station reach.

## Features

* Interactive map displaying radio station locations nationwide
* Network-based search and filtering
* Responsive and user-friendly interface
* Data integration from radio station coverage datasets

## My Contributions

As part of a development team, I was primarily responsible for:

* Integrating radio station and network data into the platform
* Implementing network search functionality
* Assisting with map data organization and visualization features

## Technologies Used

* JavaScript
* HTML/CSS
* Leaflet.js
* Papa Parse
* CSV Data Processing

This project was developed in collaboration with P48 Marketing to provide an accessible tool for exploring radio network coverage across the United States.
